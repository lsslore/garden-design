# Proyecto Pre-Entrega HTML-CSS de Talento-Tech

Este proyecto es una página web responsiva inspirada en una tienda ficticia llamada "Garden Design" esta desarrollada como parte del curso front end de talento tech.
El objetivo es aprender html y css para luego complementar con javascript.

## Características

- Estructura semántica con etiquetas HTML5: `header`, `nav`, `main`, `section`, `footer`.
- Diseño responsivo con Flexbox y Grid.
- Sección de productos en formato de cards.
- Sección de reseñas organizada con Grid.
- Adaptado con 3 medidas Media Querys para celulares, tablets y escritorio.
- Google Fonts implementado.
- Background con degradado en el header.
- Iframe de Google Maps en la sección de ubicación.
- Footer con íconos sociales y datos legales.
